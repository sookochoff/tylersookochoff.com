# tylersookochoff.com

Personal website and blog built with Astro.

## Writing Posts

1. Create a new `.md` file in `src/content/blog/`
2. Add frontmatter:
   ```yaml
   ---
   title: "Your Post Title"
   description: "Brief description of the post"
   pubDate: 2026-02-06
   ---
   ```
3. Write your post in Markdown
4. `git push` to deploy

## Local Development

```bash
npm install
npm run dev
```
